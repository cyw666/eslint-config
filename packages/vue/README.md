# @cyw666/eslint-config-vue

## Usage

### Install

Using pnpm

```bash
pnpm add -D eslint @cyw666/eslint-config-vue
```

Using npm

```bash
npm install -D eslint @cyw666/eslint-config-vue
```

Using yarn

```bash
yarn add -D eslint @cyw666/eslint-config-vue
```

### Config .eslintrc

```json
{
  // ...
  "extends": ["@cyw666/eslint-config-vue"]
}
```
