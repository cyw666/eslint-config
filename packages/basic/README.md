# @cyw666/eslint-config-basic

## Usage

### Install

Using pnpm
```bash
pnpm add -D eslint @cyw666/eslint-config-basic
```

Using npm
```bash
npm install -D eslint @cyw666/eslint-config-basic
```

Using yarn
```bash
yarn add -D eslint @cyw666/eslint-config-basic
```

### Config .eslintrc

```json
{
  // ...
  "extends": ["@cyw666/eslint-config-basic"]
}
```
